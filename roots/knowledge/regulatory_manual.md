# 障害児通所支援 規制要件マニュアル

本ドキュメントは、保育所等訪問支援マニュアル、障害児通所支援マニュアル、および東京都の報酬体系調査報告書に基づき、システム実装に必要な規制要件をまとめたものである。

---

## 1. サービス区分と対象

| サービス種類 | 対象年齢 | 概要 |
|-------------|---------|------|
| 児童発達支援 | 未就学児 | 基本動作の指導、知識技能の付与、集団生活への適応訓練 |
| 放課後等デイサービス | 就学児（6-18歳） | 学校授業終了後・休業日の生活能力向上訓練 |
| 保育所等訪問支援 | 未就学・就学児 | 専門スタッフが保育所や学校を訪問 |
| 居宅訪問型児童発達支援 | 重度障害児等 | 通所困難な児童の居宅を訪問 |

## 2. 人員配置基準（配置基準）

### 必須職種と資格要件（定員10名の標準事業所）

| 職種 | 配置義務 | 資格・経験要件 | 東京都特有の留意点 |
|------|---------|-------------|------------------|
| 管理者 | 1名（常勤） | 特になし（他職種との兼務可） | 実地指導の回答者 |
| 児童発達支援管理責任者（児発管） | 1名（常勤・専従） | 相談支援業務5年 or 直接支援業務5-8年（資格による） + 基礎研修 + OJT（2年） + 実践研修 | 更新研修の未受講による資格失効が多発。最重要職種 |
| 児童指導員 or 保育士 | 2名以上 | 保育士（国家資格）、児童指導員（教員免許/社会福祉士/精神保健福祉士/心理学科卒/実務経験2年以上） | 東京都は常に2名以上の配置を求める運用が強い |

### 児発管の配置要件の厳格化
- **基礎研修と実践研修の分離**: 基礎研修修了後、2年間の実務経験（OJT）を経て実践研修受講が必要
- **みなし配置**: やむを得ない場合、1年間に限り「みなし児発管」配置可能（届出必要、減算リスクあり）

### 常勤換算の注意事項
- 東京都は営業時間を通じて常時2名以上配置を要求
- 休憩時間を考慮すると、常勤換算で2.5-3名の確保が現実的
- 送迎で職員が外出しても基準人員を割ることは不可

## 3. 設備基準

### 指導訓練室の面積基準（2.47平方メートルルール）
- **計算式**: 利用定員 x 2.47平方メートル = 必要有効面積
- 例: 定員10名 → 24.7平方メートル以上
- 有効面積は壁の内側で計測（柱、造り付け棚、手洗い場の面積は除外）
- 定員10名の場合、全体延床面積60-80平方メートル程度が必要

### 必須設備
- **指導訓練室**: 採光・換気確保、死角のないレイアウト
- **相談室**: プライバシー配慮（個室またはパーテーション）
- **事務室**: 鍵付き書庫（個人情報保護用）
- **洗面・トイレ**: 児童の手洗い・排泄介助に適した広さ

### 建築基準法・消防法
- 床面積200平方メートル超: 用途変更手続き必要
- 消防区分: 児童福祉施設は「6項ハ」
- 自動火災報知設備・誘導灯の設置義務
- 二方向避難の確保（事実上の義務）

## 4. 報酬算定の基本構造

### 基本算定式

```
給付費 = (基本報酬単位 + 加算単位 - 減算単位) x 地域区分単価
```

### 東京都の地域区分単価

| 地域区分 | 上乗せ割合 | 1単位単価 | 適用自治体 |
|---------|-----------|----------|-----------|
| 1級地 | 20% | 11.20円 | 特別区（23区） |
| 2級地 | 16% | 10.90円 | 武蔵野市、調布市、町田市、小金井市、小平市、日野市等 |
| 3級地 | 12% | 10.61円 | 八王子市、立川市、三鷹市、府中市、昭島市等 |
| 4級地 | 10% | 10.42円 | あきる野市 |
| その他 | 0% | 10.00円 | 瑞穂町、日の出町等 |

### 児童発達支援の基本報酬（定員10名以下）

| 時間区分 | 時間要件 | 単位数 |
|---------|---------|--------|
| 区分1 | 30分以上 1時間30分以下 | 574単位 |
| 区分2 | 1時間30分超 3時間以下 | 609単位 |
| 区分3 | 3時間超 5時間以下 | 666単位 |
| 区分4 | 5時間超 | 延長支援加算と組み合わせ |

### 放課後等デイサービスの基本報酬（定員10名以下）

| 時間区分 | 時間要件 | 単位数 | 適用日 |
|---------|---------|--------|-------|
| 区分1 | 30分以上 1時間30分以下 | 466単位前後 | 全日 |
| 区分2 | 1時間30分超 3時間以下 | 480単位前後 | 全日 |
| 区分3 | 3時間超 5時間以下 | 604単位前後 | 学校休業日のみ |

## 5. 加算体系

### 人員配置系加算

| 加算名 | 単位数/日 | 要件 |
|--------|----------|------|
| 児童指導員等加配加算(I) | 187単位（常勤専従） | PT/OT/ST/保育士/児童指導員、経験5年以上、常勤換算1.0以上 |
| 児童指導員等加配加算(II) | 152単位（常勤専従） | 同資格、経験5年未満 |
| 児童指導員等加配加算(III) | 90単位 | 資格不問 |
| 専門的支援実施加算 | 150単位 | PT/OT/ST/公認心理師が直接支援 |

### 家族支援系加算

| 加算名 | 単位数/回 | 要件 |
|--------|----------|------|
| 家族支援加算(I) | 300単位 | 居宅訪問（1時間以上）。月2回限度 |
| 家族支援加算(II) | 200単位 | 居宅訪問（1時間未満） |
| 家族支援加算(III) | 100単位 | 事業所内対面 |
| 家族支援加算(IV) | 80単位 | オンライン |

### 連携系加算

| 加算名 | 単位数/回 | 要件 |
|--------|----------|------|
| 関係機関連携加算(I) | 200単位 | 保育所/学校/児相等との会議参加 |
| 関係機関連携加算(II) | 200単位 | 就学先/就職先等との連携調整 |
| 事業所間連携加算 | 150単位 | 他事業所との支援内容調整 |

### その他主要加算

| 加算名 | 単位数 | 要件 |
|--------|-------|------|
| 個別サポート加算(I) | 100-120単位/日 | ケアニーズの高い児童 |
| 個別サポート加算(II) | 125単位/日 | 要保護・要支援児童 |
| 送迎加算 | 54単位/回（片道） | 居宅-事業所間の送迎 |
| 延長支援加算 | 61-123単位 | 営業時間超の支援 |
| 入浴支援加算 | 80単位/日 | 入浴介助 |
| 食事提供加算 | 30単位/日 | 低所得世帯への食事提供 |

### 保育所等訪問支援の加算

| 加算名 | 単位数/回 | 要件 |
|--------|----------|------|
| 基本報酬 | 1,071単位 | R6改定で増額 |
| 訪問支援員特別加算(I) | 850単位 | 障害児支援業務10年以上 or 訪問支援5年以上 |
| 訪問支援員特別加算(II) | 700単位 | 障害児支援業務5年以上 or 訪問支援3年以上 |
| 多職種連携支援加算 | 200単位 | 異なる職種の複数人で訪問 |

### 処遇改善加算（全サービス共通）
- **算定方法**: （基本報酬 + 各種加算） x 加算率
- **区分**: I-IVの4区分
- **東京都の加算率**: 児童発達支援(I) 約8.1-14%、放課後等デイサービス(I) 約8.4%
- 全額を職員の賃金改善に充てること（法人利益へのプールは禁止）

## 6. 減算リスク

### 主要な減算規定

| 減算項目 | 減算率 | 要件 |
|---------|--------|------|
| BCP未策定減算 | 所定単位数の1% | 感染症/自然災害BCPが未策定、研修・訓練未実施 |
| 虐待防止措置未実施減算 | 所定単位数の1% | 虐待防止委員会未開催、指針未整備、研修未実施 |
| 身体拘束廃止未実施減算 | 所定単位数の1% | 指針未整備、委員会未開催 |
| 支援計画未作成減算 | 30%（2ヶ月目まで）/ 50%（3ヶ月目以降） | 個別支援計画が未作成 |
| 自己評価結果等未公表減算 | x85/100 | 東京都情報公表システムでの更新怠り |
| 支援プログラム未公表減算 | - | 5領域に基づく支援プログラムの未公表 |
| 定員超過減算 | x70/100 | 定員x150%超過 or 3ヶ月平均定員x105%超過 |
| 人員欠如減算 | - | 常勤換算値が基準を下回る |

## 7. 届出・申請期限

### 指定申請のスケジュール（東京都）

| 時期 | アクション |
|------|----------|
| X-5ヶ月以前 | 指定協議説明会への参加 |
| X-4ヶ月 | 事前相談・協議（図面・人員体制案） |
| X-3ヶ月 | 物件契約・内装工事・消防届出 |
| X-2ヶ月末 | **本申請（指定申請書提出）** ※1日でも遅れると翌月回し |
| X-1ヶ月 | 現地確認（実地検査） |
| X月1日 | 指定（開設） |

### 加算届出の期限
- **前月15日まで**: 加算の新規算定・区分変更の届出期限
- 例: 2月1日算定開始 → 1月15日（閉庁日は直前の開庁日）までに必着
- 東京都指定のExcel様式（体制等状況一覧表、体制届）の使用が義務

### 個別支援計画のPDCAサイクル
1. アセスメント → 2. 原案作成（5領域網羅） → 3. 担当者会議 → 4. 説明・同意（署名・捺印） → 5. モニタリング（6ヶ月に1回以上）

## 8. システム実装上の必須マスタテーブル

### 事業所マスタ
- 事業所ID、サービス種別コード、地域区分コード、定員規模、人員配置区分

### 職員マスタ
- 職員ID、保有資格、資格取得日、障害児支援業務開始日（経験年数計算用）、常勤/非常勤、専従/兼務

### 利用者（児童）マスタ
- 受給者証番号、障害支援区分、生年月日、医療的ケアスコア、強度行動障害スコア、通所受給日数上限、負担上限月額、世帯所得区分

### 単位数マスタ（バージョン管理付）
- サービスコード、適用開始日、適用終了日、単位数、算定単位（日/回/月）

### カレンダーマスタ
- 日付、学校営業日/休業日フラグ（放デイ区分判定用）、祝日フラグ

## 9. 重要なバリデーションロジック

- **定員超過チェック**: 当日利用者数が定員x150%超過 or 過去3ヶ月平均で定員x105%超過 → 全利用者の基本報酬をx70/100
- **人員欠如チェック**: 出勤職員の常勤換算値が基準以下 → 減算発動
- **重複算定禁止チェック**: 相性の悪い加算の同時算定防止
- **回数制限チェック**: 家族支援加算（月4回）、関係機関連携加算（月1回等）の上限管理
- **送迎時間の除外**: 送迎時間はサービス提供時間に含めない（令和6年改定）
- **平日/休業日判定**: カレンダーマスタで放デイの時間区分を制御（平日は区分2まで）
